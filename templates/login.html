MLN-Dashboard (ver 1.0) is an interactive dashboard to generate, analyze, and visualize complex data sets using multilayer networks. This is built as part of a CCRI NSF-funded project. Salient features are:

  • User Workspace: Users can register, securely log in, and concurrently work in their independent workspace over multiple sessions.

  • Applications: Applications based on real-world data sets like DBLP, IMDb, Accident, Airline, and Census have been pre-loaded. For security reasons, the user capability of uploading new data sets is currently disabled.

  • MLN Layer Generation: Configuration files with specific requirements (e.g., selected feature, similarity metric, thresholds) to generate layers (graphs)

  • MLN Analysis: Configuration files specify the analysis (community, centrality, …) to be performed on the generated layers.

  • Visualization: Interactive visualization alternatives, such as networks and word cloud are currently supported.
